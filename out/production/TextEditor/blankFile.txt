Pressing The Down Arrow When clicked the down arrow to take the cursor from one line  to another (or the up array) take what the length of the line should be from the size of the window then move through the linkedlist calculating the width of each character, when it is to enter a new line then take away how much we have moved from the total window length and this is how long to go over for the new line Word WrapText is stored within the linkedList and each line is tracked using an array dequeFor word wrap when the text reaches the end of the line measured by the window length, the editor scans for the last made space and word wraps from there
